JavaScript Helper

console.log()

The console.log() method is used to print messages to the console.

console.log('Hello, world!');

Variables

Variables are used to store data values in JavaScript.

var name = 'John';
var age = 25;

Primitive Types

JavaScript has several primitive types, including string, number, boolean, null, and undefined.

var name = 'John';
var age = 25;
var isStudent = true;
var nothing = null;
var something = undefined;

Logical Comparison Operators

JavaScript has logical comparison operators such as ==, !=, ===, !==, >, <, >=, and <=.

var x = 5;
var y = 10;
console.log(x < y); // Output: true

Conditional Statements

Conditional statements allow you to perform different actions based on different conditions.

var age = 20;
if (age >= 18) {
  console.log('You are an adult.');
} else {
  console.log('You are a minor.');
}

Arrays

Arrays are used to store multiple values in a single variable.

var numbers = [1, 2, 3, 4, 5];
console.log(numbers[2]); // Output: 3

Iteration

Iteration allows you to repeat a block of code multiple times.

for (var i = 0; i < 5; i++) {
  console.log(i);
}

Functions

Functions are reusable blocks of code that can be called to perform a specific task.

function greet(name) {
  console.log('Hello, ' + name + '!');
}
greet('John');

Scope

Scope determines the accessibility or visibility of variables in JavaScript.

var x = 10;

function printNumber() {
  var y = 5;
  console.log(x + y);
}

printNumber(); // Output: 15
console.log(y); // Error: y is not defined

Methods

Methods are functions that are associated with objects.

var person = {
  name: 'John',
  age: 25,
  greet: function() {
    console.log('Hello, my name is ' + this.name + '.');
  }
};

person.greet(); // Output: Hello, my name is John.

Objects

Objects are used to store multiple values as properties and methods.

var person = {
  name: 'John',
  age: 25,
  isStudent: true
};

Object Methods

Object methods are functions that are part of an object.

var person = {
  name: 'John',
  age: 25,
  greet: function() {
    console.log('Hello, my name is ' + this.name + '.');
  }
};

person.greet(); // Output: Hello, my name is John.

Object "this"

The this keyword refers to the current object, and it is used to access properties and methods within an object.

var person = {
  name: 'John',
  age: 25,
  greet: function() {
    console.log('Hello, my name is ' + this.name + '.');
  }
};

person.greet(); // Output: Hello, my name is John.

Window Object

The window object represents the browser's window and provides access to various properties and methods.

console.log(window.innerWidth); // Output: the inner width of the browser window

Traverse the DOM

You can traverse the Document Object Model (DOM) to access and manipulate HTML elements.

var element = document.getElementById('myElement');
console.log(element.innerHTML); // Output: the HTML content of the element with id "myElement"

Setting Attributes

You can set attributes of HTML elements using JavaScript.

var element = document.getElementById('myElement');
element.setAttribute('class', 'highlighted');

Create and Append Elements

You can create new HTML elements and append them to the DOM.

var newElement = document.createElement('div');
newElement.innerHTML = 'New element';
document.body.appendChild(newElement);

Timers and Intervals

JavaScript provides timers and intervals for executing code at specific time intervals.

setTimeout(function() {
  console.log('Delayed message');
}, 2000);

setInterval(function() {
  console.log('Repeated message');
}, 1000);

Event Listeners

You can add event listeners to HTML elements to respond to user interactions.

var button = document.getElementById('myButton');
button.addEventListener('click', function() {
  console.log('Button clicked');
});

Preventing Default

You can prevent the default behavior of an event using the preventDefault() method.

var link = document.getElementById('myLink');
link.addEventListener('click', function(event) {
  event.preventDefault();
  console.log('Link clicked');
});

Keyboard Events

You can handle keyboard events such as key press, key down, and key up.

document.addEventListener('keydown', function(event) {
  console.log('Key pressed: ' + event.key);
});

Event Bubbling

Event bubbling refers to the propagation of an event through nested HTML elements.

document.addEventListener('click', function(event) {
  console.log('Clicked on: ' + event.target.tagName);
});

Data Attributes

Data attributes allow you to store custom data within HTML elements.

<div id="myElement" data-user-id="123"></div>
  
var element = document.getElementById('myElement');
console.log(element.getAttribute('data-user-id')); // Output: 123

Local Storage

The local storage API allows you to store data in the browser's local storage.

localStorage.setItem('username', 'John');
console.log(localStorage.getItem('username')); // Output: John

Local Storage Objects

You can store JavaScript objects in the local storage by converting them to JSON strings.

var user = {
  name: 'John',
  age: 25
};

localStorage.setItem('user', JSON.stringify(user));
var storedUser = JSON.parse(localStorage.getItem('user'));
console.log(storedUser.name); // Output: John

Local Storage Todos

You can use local storage to store and retrieve a list of to-dos.

var todos = JSON.parse(localStorage.getItem('todos')) || [];

function addTodo() {
  var todo = document.getElementById('todoInput').value;
  todos.push(todo);
  localStorage.setItem('todos', JSON.stringify(todos));
  displayTodos();
}

function displayTodos() {
  var todoList = document.getElementById('todoList');
  todoList.innerHTML = '';
  for (var i = 0; i < todos.length; i++) {
    var todoItem = document.createElement('li');
    todoItem.textContent = todos[i];
    todoList.appendChild(todoItem);
  }
}

displayTodos();
